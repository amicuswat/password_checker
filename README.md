# Проверка надежности паролей
Проверяет наличие в пароле цифр, спецсимволов, заглавных и строчных букв, а так
же длинну пароля. Выдает рейтинг надежности. Максимально надежный 12.

## Запуск

- Скачайте код
- Запустите командой:
```
python main.py
```
